---
description: Commit and push latest changes to GitHub
---

1. Check the current status of the repository to match the changes you expect to commit.
```bash
git status
```

2. Add all changes to the staging area.
```bash
git add .
```

3. Commit the changes. **Important**: Replace "Update" with a descriptive message about your changes.
```bash
git commit -m "Update"
```

4. Push the changes to the remote repository.
```bash
git push
```
